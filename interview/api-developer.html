Sample Interview Questions

Q1
How do you design a RESTful API?
Tip: Show resource modeling, HTTP method semantics, status codes, pagination, and HATEOAS principles.
Q2
What is your experience with GraphQL?
Tip: Discuss schema design, resolvers, query optimization, N+1 prevention, and when to choose GraphQL over REST.
Q3
How do you handle API versioning?
Tip: Discuss URL versioning, header versioning, content negotiation, and deprecation strategies.
Q4
What is your approach to API documentation?
Tip: Mention OpenAPI/Swagger, interactive docs, code generation, and keeping docs in sync with implementation.
