Sample Interview Questions

Q1
How do you approach selecting a model architecture for a new problem?
Tip: Discuss the problem type (classification, generation, etc.), data size, latency requirements, and trade-offs between custom models vs. fine-tuning pretrained ones.
Q2
Explain how you handle data quality issues in a machine learning pipeline.
Tip: Show experience with data cleaning, handling missing values, class imbalance, feature engineering, and automated data validation checks.
Q3
How do you deploy and monitor AI models in production?
Tip: Discuss CI/CD for ML, A/B testing, model versioning, drift detection, latency monitoring, and rollback strategies.
Q4
Describe a project where you used large language models or transformers.
Tip: Use STAR. Show understanding of prompting, fine-tuning, RAG, or RLHF and how you evaluated model output quality.
